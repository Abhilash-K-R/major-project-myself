PROJECT OVERVIEW
================================================================================
A production-ready React blog frontend application built with TypeScript, Vite,
and Tailwind CSS. Designed for dark-mode-first user experience with seamless
integration to a local Express/SQLite backend at http://localhost:5000/api.

The application provides a complete blog platform with:
  â€¢ Blog post listing with search and filtering
  â€¢ Post creation with form validation
  â€¢ Post detail view with metadata
  â€¢ Post deletion with confirmation
  â€¢ Dark/light mode toggle
  â€¢ Responsive design (mobile & desktop)
  â€¢ TanStack Query for async state management
  â€¢ Toast notifications for user feedback


TECHNOLOGY STACK
================================================================================
Frontend Framework:   React 18.3.1 + TypeScript
Build Tool:          Vite
Styling:             Tailwind CSS + next-themes (dark mode)
Routing:             React Router v6
